# Backend - Guess The Movie

Node.js backend API for the movie guessing game.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

## Available Scripts

- `npm run start` - Start the server (runs on port 5001)
- `npm run dev` - Start with nodemon for development
- `npm run format` - Format code with Prettier

## Running the Server

```bash
node src/index.js
```

The server will run on port 5001.
